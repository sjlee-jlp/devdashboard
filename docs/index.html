EUC Dev Dashboard Automation 

This script has been written by Simon Lee for the automation of the EUC Dev Dashboard.

Features

  • Manual archiving of data from the Current Week sheet
  • Automatic emailing when projects are due/overdue

Tech

This script uses the following technologies to work properly:

  • Google Apps Script - scripting language for light-weight application development.

Set Up

The application requires a Google Account to run.

First Run
  • Select the ‘Configuration and Help’ menu from the toolbar
  • Select ‘Authorise’

The script is now authorised and you can interact with it

Email Alert Configuration
  • Visit the Triggers Page
  • Select Add Trigger
  • From the ‘Choose which function to run’ drop-down, select: sendAlert
  • From the ‘Select event source’ drop-down, select: Time-driven
  • From the ‘Select type of time based trigger’ drop-down, select: Week timer
  • From the ‘Select day of the week’ drop-down, select the day of the week that you want the alert to be sent
  • From the ‘Select time of day’ drop-down, select a time of the day that you want the alert to be sent.
  • Select Save

Manual Archiving Instruction

To perform a manual archive of the data from the Current Week sheet, do the following:

  • Select the Configuration and Help menu from the toolbar.
  • Select Manual Archive
